Lección 2: Tablas

Para realizar una tabla es necesario conocer 4 etiquetas sintacticas:

table: Esta etiqueta contiene la totalidad de la tabla, todo lo que queramos que se incluya en la tabla, debera ponerse dentro de esta etiqueta

tr: esta etiqueta significa fila de tabla, permite la creación de filas, es importante entender, que a menos que coloquemos una nueva tr, la tabla rellenara las celdas siguiendo la fila, hasta que todas sus columnas sean completadas

th: Esta etiqueta significa titulo de tabla, se usa para añadir un titulo a nuestra tabla, tiene el mismo efecto que la etiqueta h1,2 y 3

td: Esta etiqueta significa datos de tabla, se usa para rellenar las celdas de la tabla

Es esencial entender que a la hora de realaizar uan tabla con html, no debemos pensar en ella a modo de filas y columnas como si se rellenara librememente un excel,dado quee stamos relaizando la actividad con un lenguaje de marcas y no con una interfaz grafica.
